<!DOCTYPE>

Es la etiqueta de inicio de todos los archivos html, se utiliza para informar al navegador del tipo de documento. Da estructura a la página 


            <!DOCTYPE html>
            <html>
                <head>
                    <title>Titulo</title>
                </head>
                <body>
                </body>
            </html>
