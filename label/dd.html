<dd>

Esta etiqueta se utiliza para dar una definición, debe ir dentro de la etiqueta <dl> y acompañada de <dt> para dar el término que se está definiendo.

Sintaxis

<dl>

<dt>HTML</dt>

<dd>Lenguaje de marcado que nos permite indicar la estructura de nuestro documento mediante etiquetas.</dd>

<dt>CSS</dt>

<dd>Lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.</dd>

<dt>JavaScript</dt>

<dd>Lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico.</dd>

</dl>

Ejemplo

HTML
Lenguaje de marcado que nos permite indicar la estructura de nuestro documento mediante etiquetas.
CSS
Lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.​
JavaScript
Lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, ​ basado en prototipos, imperativo, débilmente tipado y dinámico.
