<a>

  • href
  • hreflang
  • media
  • ping
  • referrerpolicy
  • rel
  • target
  • type

La etiqueta <a> define un hipervínculo que se utiliza para vincular un archivo a nuestra página.

El atributo más importante del elemento <a> es el atributo href, que indica el destino del enlace.

href

El valor del atributo href da la ruta absoluta o relativa del archivo al que se realiza el hipervínculo.

Sintaxis

<a href="./icon.png">Click aquí para ver el icono</a>

Ejemplo

Click aquí para ver el icono

hreflang

El atributo hreflang Indica el idioma del archivo o url del hipervínculo.

Sintaxis

<a href="./English.pdf" hreflang="en"></a>

Ejemplo

Click aquí para ver el PDF

media

El atributo media especifica para qué medio o dispositivo está optimizado el documento vinculado. Este atributo se utiliza para especificar que la URL de destino está diseñada para dispositivos especiales (como iPhone), medios impresos o de voz.

Este atributo es muy extenso, por lo tanto aqui tienes unas tablas para poder ver todos los posibles valores y como configurarlo de la forma mas óptima posible.

Dispositivos
Valor Descripción
all Por defecto. Apto para todos los dispositivos
aural Sintetizadores de voz
braille Dispositivos de retroalimentación en braille
handheld Dispositivos de mano (pantalla pequeña, ancho de banda limitado)
projection Proyectores
print Modo de vista previa de impresión
screen Pantalla de ordenador
tty Teletipos y medios similares que utilizan una cuadrícula de caracteres de tono fijo
tv Dispositivos de tipo televisión (baja resolución, capacidad de desplazamiento limitada)
Condiciones
Valor Descripción
width Especifica el ancho del área de visualización de destino. Se pueden utilizar los prefijos "min-" y "max-".
Ejemplo: media="screen and (min-width:500px)"
height Especifica la altura del área de visualización de destino. Se pueden utilizar los prefijos "min-" y "max-".
Ejemplo: media="screen and (max-height:700px)"
device-width Especifica el ancho de la pantalla/página de destino. Se pueden utilizar los prefijos "min-" y "max-".
Ejemplo: media="screen and (device-width:500px)"
device-height Especifica la altura de la pantalla/página de destino. Se pueden utilizar los prefijos "min-" y "max-".
Ejemplo: media="screen and (device-height:500px)"
orientation Especifica la orientación de la pantalla/página de destino. Valores posibles: "portrait" u "landscape".
Ejemplo: media="all and (orientation: landscape)"
aspect-ratio Especifica la relación ancho/alto del área de visualización de destino. Se pueden utilizar los prefijos "min-" y "max-".
Ejemplo: media="screen and (aspect-ratio:16/9)"
device-aspect-ratio Especifica la relación del ancho/alto del dispositivo de la pantalla/página de destino. Se pueden utilizar los prefijos "min-" y "max-".
Ejemplo: media="screen and (aspect-ratio:16/9)"
color Especifica los bits por color de la pantalla de destino. Se pueden utilizar los prefijos "min-" y "max-".
Ejemplo: media="screen and (color:3)"
color-index Especifica el número de colores que puede manejar la pantalla de destino. Se pueden utilizar los prefijos "min-" y "max-".
Ejemplo: media="screen and (min-color-index:256)"
monochrome Especifica los bits por píxel en un búfer de fotogramas monocromáticos.Se pueden utilizar los prefijos "min-" y "max-".
Ejemplo: media="screen and (monochrome:2)"
resolution Especifica los bits por píxel en un búfer de fotogramas monocromáticos. Se pueden utilizar los prefijos "min-" y "max-".
Ejemplo: media="screen and (monochrome:2)"
scan Especifica el método de escaneo de una pantalla de televisión. Los valores posibles son "progressive" y "interlace"
Ejemplo: media="print and (resolution:300dpi)"
grid Especifica si el dispositivo de salida es de cuadrícula o mapa de bits. Los valores posibles son "1" para la cuadrícula y "0" en caso contrario.
Ejemplo: media="handheld and (grid:1)"
Operadores
Valor Descripción
and Especifica un operador AND
not Especifica un operador NOT
, Especifica un operador OR

Sintaxis

<a href="./index.html" media="all screen and (max-width: 800px)"></a>

Ejemplo

Abre la página de Inicio indicandole que todas las pantallas son compatibles, inclusive las menores a 800px de ancho

ping

El atributo ping especifica una lista de URLs que se notificarán si el usuario sigue el hipervínculo. Cuando el usuario hace click en el hipervínculo, el atributo ping enviará una breve solicitud HTTP POST a la URL especificada.

Este atributo es útil para monitorear o rastrear la navegación de los usuarios.

Sintaxis

<a href="./index.html" ping="./tracks.txt"></a>

Ejemplo

Manda la URL solicitada a tracks.txt

referrerpolicy

El atributo referrerpolicy especifica qué información de referencia se envía cuando el usuario hace click en el hipervínculo.

Este atributo puede aceptar los siguientes valores:

Valores
Valor Descripción
no-referrer No se envía información de referencia
no-referrer-when-downgrade Por defecto. Envía el origen, la ruta y la cadena de consulta si el nivel de seguridad del protocolo permanece igual o es más alto (HTTP a HTTP, HTTPS a HTTPS, HTTP a HTTPS está bien). No envía nada a un nivel menos seguro (HTTPS a HTTP no está bien)
origin Envía el origen (esquema, host y puerto) del documento.
origin-when-cross-origin Envía el origen del documento para la solicitud de origen cruzado. Envía el origen, la ruta y la cadena de consulta para la solicitud del mismo origen.
same-origin Envía una referencia para una solicitud del mismo origen. No envía ninguna referencia para la solicitud de origen cruzado
strict-origin-when-cross-origin Envía el origen si el nivel de seguridad del protocolo sigue siendo el mismo o es más alto (HTTP a HTTP, HTTPS a HTTPS y HTTP a HTTPS está bien). No envía nada a un nivel menos seguro (HTTPS a HTTP)
unsafe-url Envía el origen, la ruta y la cadena de consulta (independientemente de la seguridad). ¡Utilice este valor con cuidado!

Sintaxis

<a href="./index.html" referrerpolicy="no-referrer"></a>

Ejemplo

Abre la página de Inicio sin hacer referencia a ninguna información.

rel

El atributo rel especifica la relación entre el documento actual y el documento vinculado. Solo se usa si el atributo href está presente.

Los motores de búsqueda pueden utilizar este atributo para obtener más información sobre un enlace.

Valores
Valor Descripción
alternate Proporciona un enlace a una representación alternativa del documento (es decir, página impresa, traducida o reflejada)
author Proporciona un enlace al autor del documento.
bookmark URL permanente utilizada para los marcadores
external Indica que el documento referenciado no es parte del mismo sitio que el documento actual
help Proporciona un enlace a un documento de ayuda.
license Proporciona un enlace a la información de licencia del documento.
next Proporciona un enlace al siguiente documento de la serie.
nofollow Vínculos a un documento no endosado, como un vínculo pago.
noopener Requiere que cualquier contexto de navegación creado siguiendo el hipervínculo no debe tener un contexto de navegación de apertura
noreferrer Hace que la referencia sea desconocida. No se incluirá ningún encabezado de referencia cuando el usuario haga clic en el hipervínculo
prev El documento anterior en una selección
search Enlaces a una herramienta de búsqueda del documento
tag Una etiqueta (palabra clave) para el documento actual

Sintaxis

<a href="https://www.google.es/" rel="external"></a>

Ejemplo

Abre la página de Google indicando que es un enlace externo

target

El atributo target especifica como abrir el documento vinculado.

Puede tener los siguientes valores:

Valores
Valor Descripción
_blank Abre la página vinculada en una nueva ventana o pestaña
_self Abre la página vinculada en la misma ventana o pestaña (este es el valor predeterminado)
_parent Abre la página vinculada en la ventana o pestaña anterior
_top Abre la página vinculada en todo el cuerpo de la ventana o pestaña

Sintaxis

<a href="./index.html target="_blank"></a>

Ejemplo

Abre la página de Inicio en una nueva pestaña

type

El atributo type se usa para indicar el tipo de archivo vinculado.

A continuación dejamos una tabla con unos pocos de ejemplos de los tipos de archivo más usados:

Valores
Valor Tipo de archivo
text/html Documento HTML
video/mp4 Archivo de Video MP4
audio/mp3 Archivo de Audio MP3
img/png Imagen PNG

Sintaxis

<a href="./index.html" type="text/html"></a>

Ejemplo

Abre la página de Inicio indicando que es código html
