<a>

  • href
  • hreflang
  • media
  • ping
  • referrerpolicy
  • rel
  • target
  • tipe

La etiqueta <a> define un hipervínculo, que se utiliza para vincular de una página a otra. El atributo más importante del <a>elemento es el atributo href="./index.html", que indica el destino del enlace.

href

El valor href="" dentro de <a> da la ruta absoluta o relativa del archivo al que se realiza el hipervinculo. 

<a href="/images/icone.png">Clik aquí para ver el icono</a>

Ejemplo:

Clik aquí para ver el icono

hreflang

Indica el idioma del archivo o url del hipervínculo.

Sintaxis

<a href="/pdf/English.pdf" hreflang="en">Clik aquí para ver el PDF</a>

Ejemplo:

Clik aquí para ver el PDF
